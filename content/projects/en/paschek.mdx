#  Project 1 — PascheK-ChatBox (Electron + Next.js + AI)

###  Goal  
Create a full desktop application that lets you:  
- chat with an AI model,  
- analyse code, files and entire projects,  
- manage a local vector database (RAG),  
- provide a modern, cross-platform UX.

###  Features  
- Desktop app built with **Electron + Next.js**  
- Automatic analysis of files / projects  
- Document reading and embeddings  
- Local RAG with Drizzle + Neon  
- Multiple conversation contexts  
- Modern, responsive UI with animations  
- macOS build via Electron Builder  

###  Stack  
- Electron  
- Next.js  
- TypeScript  
- OpenAI / AI-SDK  
- Drizzle ORM  
- Neon / SQLite  
- Tailwind CSS  

###  Repo  
- https://github.com/PascheK/PascheK-ChatBox

#  Project 3 — Homelab (Personal Dell R730 Infrastructure)

###  Goal  
Maintain a complete infrastructure to host my projects, personal services and development environments.

###  Hosted stack  
- **Dell PowerEdge R730** running Ubuntu Server  
- Docker + docker-compose  
- Reverse Proxy (Traefik or Nginx Proxy Manager)  
- Full Nextcloud instance  
- Databases (MariaDB / PostgreSQL)  
- Internal services (VPN, storage, monitoring)

### ⚙ Features  
- Hosting of personal services  
- Isolated Docker networks  
- Automatic SSL certificates  
- Regular backups  
- Monitoring + logs  
- Performance and high availability  

###  Skills involved  
- DevOps  
- Security  
- Self-hosting  
- Docker networking  
- Reverse proxy & SSL  
- Advanced Linux administration
