## Overview

Beryl Impact Scroll Storytelling is a Next.js experience that guides the audience through the before, during, and after phases of Hurricane Beryl using scroll-synced videos, timed overlays, and dedicated narrative sections.

## Features

- Scroll-based storytelling across three phases with centralized overlay configuration  
- Immersive hero with animated text and visual scroll cues  
- Smooth scrolling with Lenis, GSAP ScrollTrigger triggers, and a global overlay for orchestrated transitions  
- StoryMap and ArcGIS dashboard integrations, completed with downloadable PDF reports  
- Preloading of key videos to stabilize scroll-driven calculations before rendering  

## Tech Stack

- Next.js (App Router) and TypeScript  
- React 19  
- Tailwind CSS 4 design system  
- GSAP ScrollTrigger and Framer Motion animations  
- Smooth scrolling with Lenis  
- Lucide Icons for visual markers  

## Role & Contribution

As a front-end storytelling developer:  
- Compose Next.js pages, integrate GSAP/Framer Motion animations, and maintain Lenis-based smooth navigation  
- Structure overlay data, ensure timing consistency, and enhance configurable phase content  
- Maintain StoryMap/ArcGIS integrations and downloadable resources to distribute insights
