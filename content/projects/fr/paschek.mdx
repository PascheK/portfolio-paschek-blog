#  PascheK – Projets personnels

Voici une sélection de mes projets personnels les plus aboutis, représentant mes compétences en développement web, IA, architecture logicielle et DevOps.  
Ces projets sont développés en autonomie, en dehors de tout mandat externe.

---

#  Projet 1 — PascheK-ChatBox (Electron + Next.js + IA)

###  Objectif  
Créer une application desktop complète permettant :  
- de discuter avec un modèle IA,  
- d’analyser du code, des fichiers et des projets entiers,  
- de gérer une base vectorielle locale (RAG),  
- d’offrir une UX moderne multiplateforme.

###  Fonctionnalités  
- App desktop **Electron + Next.js**  
- Analyse automatique de fichiers / projets  
- Lecture et embeddings de documents  
- RAG local avec Drizzle + Neon  
- Multi-contextes de discussion  
- UI moderne, responsive, animations  
- Build macOS via Electron Builder  

###  Stack  
- Electron  
- Next.js  
- TypeScript  
- OpenAI / AI-SDK  
- Drizzle ORM  
- Neon / SQLite  
- Tailwind CSS  

###  Repo  
- https://github.com/PascheK/PascheK-ChatBox

#  Projet 3 — Homelab (Infrastructure personnelle Dell R730)

###  Objectif  
Maintenir une infrastructure complète pour héberger mes projets, mes services personnels et mes environnements de développement.

###  Stack hébergé  
- **Dell PowerEdge R730** sous Ubuntu Server  
- Docker + docker-compose  
- Reverse Proxy (Traefik ou Nginx Proxy Manager)  
- Nextcloud complet  
- Bases de données (MariaDB / PostgreSQL)  
- Services internes (VPN, stockage, monitoring)

### ⚙ Fonctionnalités  
- Hébergement de services personnels  
- Réseau Docker isolé  
- Certificats SSL automatiques  
- Sauvegardes régulières  
- Monitoring + logs  
- Performance et haute disponibilité  

###  Compétences mobilisées  
- DevOps  
- Sécurité  
- Self-hosting  
- Docker networking  
- Reverse proxy & SSL  
- Administration Linux avancée
