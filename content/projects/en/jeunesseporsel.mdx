## Overview

The Jeunesse de Porsel website is a Next.js showcase centralizing information on theater, fondue, and contacts for the youth society.

## Features

- Responsive navigation with desktop header, mobile sidebar, and CTA buttons
- Theater section with synopsis, schedule, actor portraits, and booking instructions
- Fondue module with dynamic events, photo carousel, and OTP-secured registration flows
- Reusable Embla-powered carousel for galleries and actor profiles
- Contact form with Nodemailer integration for message delivery and confirmation

## Tech Stack

- Next.js 15 and React 19 RC (App Router)
- TypeScript, Zod, and React Hook Form for type-safe forms
- Tailwind CSS, Radix UI primitives, and Embla Carousel
- Nodemailer and Vercel Analytics for communications and observability

## Role & Contribution

As a full-stack Next.js developer:

- Enhance immersive sections, factorize carousels, and improve OTP-secured forms
- Maintain API connectors and provide smooth user feedback with toasts
